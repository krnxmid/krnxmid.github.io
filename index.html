Projects

Modbus data over Website

Displaying modbus data on a website using python and fastapi. this project include EC2 and a G1 mini computer to accomplish this task. the G1 mini computer sends the modbus data in the form of POST requests and the EC2 handles the requests and displays it in a tabular form. This dashboard has the functionality of downloading the data as an excel file and also supports databasa meaning the data will be stored and can be accessed anytime.

Wifi deauther using ESP32

I've created many esp-32 deauthers in the past and they are very fun! its pretty easy to do so, you can get the full documentation of deauther.com and all you need is an esp32 and a usb micro cable for flashing, and thats it! now you can access the deauther web interface on a local host and select the wifi to attack.
