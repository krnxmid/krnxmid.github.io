Projects

Modbus Data over Website

Displaying modbus data on a website using Python and FastAPI. This project includes EC2 and a G1 mini computer to accomplish this task. The G1 mini computer sends the modbus data in POST requests, and EC2 handles the requests and displays them in a tabular form. This dashboard supports downloading data as an Excel file and a database to store and access data anytime.

Wifi Deauther using ESP32

I've created many ESP-32 deauthers. It's a fun project! You can follow full documentation on deauther.com, flash the ESP32, and access the deauther interface on a local host.

ESP32 CAM

If you enjoy IoT projects, the ESP32 CAM is a must-try. It's a camera with an integrated CPU, enabling color tracking, object tracking, surveillance, and more.
